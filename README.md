# MediPlus 💊

MediPlus es una botica comprometida con tu bienestar, donde encontrarás medicamentos de calidad, productos de cuidado personal y asesoría profesional en un ambiente moderno y accesible. Nuestro objetivo es brindarte atención rápida, confiable y cercana, cuidando de tu salud y la de tu familia todos los días.

---

## 🚀 Características

- Visualización de medicamentos
- Agregar nuevos medicamentos
- Backend con Node.js y Express
- Base de datos MySQL
- Frontend moderno con JavaScript
